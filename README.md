# 🔐 Password Strength Checker
A pure Python mini-project built from scratch to evaluate and validate passwords based on standard security criteria. 

##  Description
This project takes a user's password input and checks it against specific conditions to determine its strength. It was built entirely using Python's core logic and string manipulation capabilities, without relying on external libraries. It serves as a practical exercise in conditional logic and fundamental cybersecurity concepts.

##  Features / Criteria Checked
To be considered "Strong," a password must fulfill the following criteria:
* Minimum length requirement (e.g., 8+ characters)
* Contains at least one uppercase letter (A-Z)
* Contains at least one lowercase letter (a-z)
* Contains at least one number (0-9)
* Contains at least one special character (!, @, #, $, etc.)
